## Scheduling multi-stage pipelines

#### Stage 1: 
producer(x, y) = sin(x * y)

#### Stage 2: 
consumer(x, y) = (producer(x, y)   +
                  producer(x, y+1) +
                  producer(x+1, y) +
                  producer(x+1, y+1))/4
